# iOS Shortcut Recipe for Crate Studio

## One-Tap A-Shell Paste

1. Open Shortcuts app on iOS
2. Tap + to create new Shortcut
3. Add action: 'Get Clipboard'
4. Add action: 'Open App' → select A-Shell
5. Add action: 'Run Shell Script' or use clipboard paste
6. Name the Shortcut 'Crate Paste'
7. Add to Home Screen

## Usage
- In Crate Studio, tap 'DROP TO CRATE'
- Tap the 'Crate Paste' shortcut on Home Screen
- A-Shell opens and runs the script automatically

This makes the studio feel native. Works offline after first setup.
